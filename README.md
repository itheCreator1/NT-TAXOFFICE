# NT - TAXOFFICE Website

A fully responsive, accessible accounting and tax consulting firm website built with semantic HTML5, modern CSS with dark mode support, and vanilla JavaScript.

## Overview

NT - TAXOFFICE is a professional services website showcasing accounting, tax preparation, payroll, and consulting services. The site is optimized for user experience across all devices and includes accessibility features meeting modern web standards.

## Features

- Responsive design with mobile-first approach
- Dark mode support via CSS custom properties
- Accessible navigation with ARIA labels and skip links
- Smooth scroll animations using Intersection Observer API
- Client-side form validation with user feedback
- Semantic HTML5 markup
- Font Awesome icons integration
- Google Fonts typography (Poppins, Roboto)

## Project Structure

```
.
```

## Technical Details

### Styling

The stylesheet uses CSS custom properties organized into:
- Color system (light and dark mode variants)
- Spacing scale (xs through 3xl)
- Fluid typography with viewport-based scaling
- Shadow depths and border radius tokens
- Transition timing for consistent animations

### Accessibility

- Skip to main content link for keyboard navigation
- ARIA labels on interactive elements
- Semantic HTML structure
- Form validation with visual feedback
- Keyboard-accessible navigation menu

### JavaScript

The script provides:
- Mobile menu toggle functionality
- Smooth anchor link scrolling
- Contact form validation and submission handling
- Scroll-based section animations via Intersection Observer

## Browser Support

Works on all modern browsers. Gracefully degrades on older browsers with JavaScript disabled.
